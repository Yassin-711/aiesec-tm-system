# Google Sheets Structure - AIESEC TM System

## üìä Sheet Tabs Overview

1. **People** - All members and Team Leaders
2. **Team Leaders** - Summary of all TLs
3. **Meetings** - Meeting schedule with QR codes
4. **Attendance** - Attendance records from QR scans
5. **Weekly Pulse** - Weekly check-in responses
6. **Tasks** - Task assignments and performance
7. **VP TM Dashboard** - LC-wide overview
8. **iGV TL Dashboard** - iGV function dashboard
9. **oGV TL Dashboard** - oGV function dashboard
10. **OGT TL Dashboard** - OGT function dashboard
11. **IGT TL Dashboard** - IGT function dashboard
12. **BXP TL Dashboard** - BXP function dashboard
13. **B2B TL Dashboard** - B2B function dashboard
14. **Recognition** - Monthly recognition records

## 1Ô∏è‚É£ PEOPLE TAB

### Columns (A to K):

| Column | Header | Type | Formula/Notes |
|--------|--------|------|---------------|
| A | Member ID | Text | Auto-generated: `="TM"&TEXT(ROW()-1,"0000")` |
| B | Name | Text | Manual entry |
| C | Role Type | Dropdown | Options: VP TM, EB, Team Leader, Member |
| D | Function | Dropdown | Options: iGV, oGV, OGT, IGT, BXP, B2B, TM |
| E | Team Leader | Dropdown | `=IF(C2="Team Leader","N/A",FILTER(People!B:B,People!C:C="Team Leader",People!D:D=D2))` |
| F | Email | Text | Manual entry |
| G | Phone | Text | Manual entry |
| H | Start Date | Date | Manual entry |
| I | Status | Dropdown | Options: Active, On Hold, Left |
| J | Attendance % | Number | `=IFERROR(COUNTIF(Attendance!C:C,A2)/COUNTIF(Attendance!C:C,A2,"<>")*100,0)` |
| K | Pulse Score (Last 4 Weeks) | Number | `=IFERROR(AVERAGE(FILTER(Weekly Pulse!H:H,Weekly Pulse!C:C=A2,Weekly Pulse!A:A>=TODAY()-28)),0)` |
| L | Task Score | Number | `=IFERROR(SUMIF(Tasks!B:B,A2,Tasks!G:G)/COUNTIF(Tasks!B:B,A2),0)` |
| M | Engagement Status | Text | `=IF(OR(J2<60,K2<2),"RED",IF(OR(J2<80,K2<3),"YELLOW","GREEN"))` |

### Data Validation Rules:
- **Column E (Team Leader)**: Only shows TLs from same function, excludes TM function TLs for non-TM members
- **Column D (Function)**: If Role Type = "Team Leader" and Function = "TM", then Column E should show "N/A"

## 8Ô∏è‚É£ FUNCTION TL DASHBOARDS (iGV, oGV, OGT, IGT, BXP, B2B)

Each function dashboard follows the same structure:

### Header:
- Function Name
- Team Leader Name (from filter)
- Total Members: `=COUNTIF(People!E:E,"[TL Name]")`

### Metrics:
- Team Attendance %: `=AVERAGEIF(People!E:E,"[TL Name]",People!J:J)`
- Team Pulse Score: `=AVERAGEIF(People!E:E,"[TL Name]",People!K:K)`
- Tasks Completed: `=COUNTIFS(Tasks!B:B,"[Member IDs]",Tasks!G:G,"Completed")`

### Team Members Table:

| Name | Attendance % | Pulse | Tasks Score | Status |
|------|--------------|-------|-------------|--------|
| `=FILTER(People!B:B,People!E:E="[TL Name]")` | `=FILTER(People!J:J,People!E:E="[TL Name]")` | `=FILTER(People!K:K,People!E:E="[TL Name]")` | `=FILTER(People!L:L,People!E:E="[TL Name]")` | `=FILTER(People!M:M,People!E:E="[TL Name]")` |

### At-Risk Members in Function:
- List filtered by TL name and RED status
